1. Eclipse¶

1.1. Download Eclipse IDE¶

You can download Eclipse IDE from this page

  • http://www.eclipse.org/downloads/

Download a newest version of “Eclipse IDE for Java EE Developers”. Recent version now is “Indigo (3.7)”

1.2. Install Eclipse¶

Unpack the downloaded file and move eclipse folder to your home directory. In this document, we use C:\Users\<name>\eclipse-jee-indigo-win32\eclipse.

Start Eclipse by double clicking the eclipse icon.

Click OK button.

Click Finish button.

Click Workbench button.

1.3. Hello World project¶
