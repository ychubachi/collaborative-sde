8. Cygwin¶

8.1. What is Cygwin?¶

Cygwin is a collection of tools which provide a Linux look and feel environment for Windows.

See Cygwin for more information.

8.2. Installing Cygwin¶

Create a directory for Cygwin under the user’s home directory like C:\Users\<name>.

Make two directories to prepare this installation.

  1. C:\Users\<name>\Cygwin
  2. C:\Users\<name>\Cygwin\Packages

Download setup.exe to C:\Users\<name>\Cygwin.

Run setup.exe at C:\Users\<name>\Cygwin.

  • Choose “install from internet” option.
  • Set C:\Users\<name>\Cygwin to “Root directory”.
  • Set C:\Users\<name>\Cygwin\Packages to “Local Package Directory”.
  • Use default to other options.

setup.exe automatically download related packages from internet.

8.3. Run Cygwin¶

Run cygwin.bat at C:\Users\<name>\Cygwin.

8.4. Installing required packages¶

Run setup.exe again and move to “Select Packages” dialog.

Install following packages

  • python under “All/Python”
  • git under “All/Devel”

Note: Search text box will help to find these packages.
